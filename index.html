O que há em uma pàgina web?

Em uma pàgina da web, a linguagen de marcação de hipertexto (HTML) oferece seu conteúdo, as folhas de estilo em cascata (CSS) definem a aparência da página e o Java Script é usado para fornecer a interatividade com o usuário ou seja a lógica de negócios.

Atenção

Leia com cuidado o que está sendo apresentado.

O conteúdo estilo e lógica de negócios são separados em arquivos HTML, CSS e Java Script, respectivamente. Um principio de design na programação moderna é a separação de interesses. Entre as muitas razões para separar interesses, uma é a simplicidade e a simplicidade e a reutilização. Exemplo, ao estilizar os elementos usando o CSS, você pode aplicar estilos a todos os elementos de uma página independentemente da complexidade da página, em vez de codificar a aparência separadamente em cada elemento.
