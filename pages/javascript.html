Vincular o Java Script

O Java Script é uma linguagen de programação que ajuda voçê a adicionar interatividade no sua página web. Quando você seleciona um botão o Java Script é o código que define o evento ou o comportamento que ocorrerá como abrir uma janela pop up, usando o Java Script você pode adicionar e remover conteúdo como o texto da página web sem recarrega-lo Colocamos a tag do Java Script no final do
